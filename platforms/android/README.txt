Android projects : 

To use this project you have to installed android sdk and android ndk.
To execute specific android command, you must have set the path.

guido-engine-android : 
This project is used to build guido engine for android.

canvas-draw-command : 
this project build a library to draw score in android canvas.

simple-guido-editor : 
This project is a demo application to type guido code and draw it in two differents tabs :
	- in a web view with a svg representation
	- in a android canvas with canvas-draw-command library.
