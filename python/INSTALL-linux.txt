- install Qt 4.7.4 (or higher): http://doc.qt.digia.com/qt/install-x11.html

download archive from http://qt-project.org/downloads (e.g. ftp://ftp.qt-project.org/qt/source/qt-everywhere-opensource-src-4.7.4.tar.gz)
unzip
./configure
make
make install

or:
sudo apt-get install qt-sdk


- install PySide: http://qt-project.org/wiki/Building_PySide_on_Linux
use build-scripts (e.g. git clone git://github.com/PySide/BuildScripts.git pyside-git)
follow README file

or:
sudo apt-get install python-pip shiboken libshiboken-dev
pip install PySide

(or e.g. pip install PySide --install-option="--qmake=/usr/local/Trolltech/Qt-4.7.4/bin/qmake")
