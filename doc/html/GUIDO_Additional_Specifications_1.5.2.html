1. Accelerando / Ritardando

1. Accelerando / Ritardando

\accel<dx, dy, color, tempo, abstempo, font, fattrib>(notes)

\rit<dx, dy, color, tempo, abstempo, font, fattrib>(notes)

Places the text "accel." or "rit." under the staff where it occurs, and draws a dotted line to the end position.


  • Parameters (in the order of their default positions):

    • dx, dy - graphic offsets of the accelerando/ritardando

      • standard values:
        • unit parameters
      • default value: 0

    • color - graphic color of the trill

      • standard values:
        • color strings
      • default value: "black"

    • tempo - tempo for the beginning note

      • standard values:
        • string (ex : "60", "80")
      • default value: none

    • abstempo - tempo for the ending note

      • standard values:
        • string (ex : "60", "80")
      • default value: none

    • font - the font name for the text accel or rit

      • standard values:
        • string (ex : "Arial")
      • default value: "Times new Roman"

    • fattrib - the font attributes

      • standard values:
        • "b" (for bold)
        • "i" (for italic)
      • default value: none


  • Range: obligatory


  • Semantics:
    • acceleration or deceleration within a group of notes

Examples: 

[ \rit<fattrib="b">(e g b f a) \accel<tempo="60", color="red", abstempo="80", dy=22, font="Andalus">( e d e2 e3 f1 a)]
