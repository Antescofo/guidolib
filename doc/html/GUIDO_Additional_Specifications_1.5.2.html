The GUIDO Music Notation Format

Additionnal and new specifications

Supported by the GUIDO Engine version 1.5.2
Grame - October 2013

1. Meter 2. Tuplet

1. Meter

\meter<type, size, autoBarlines>

Set the meter of the voice. Meter can be complex : numerator can be a sum. See examples.


  • Parameters (in the order of their default positions):

    • type - meter type

      • standard values:
        • string
      • default value: "4/4"

    • size - graphic size of meter

      • standard values:
        • unit parameter
      • default value: 1

    • autoBarlines - Sets auto barlines on or off

      • standard values:
        • strings ("on" / "off")
      • default value: "on"


  • Semantics:
    • segmentation of music part's duration

Examples: 

[ \meter<"2+2+3/4"> a a c2 a1 e2 e d c ]
[ \meter<"C"> a b c d e ]

2. Tuplet

\tuplet<format, dy1, dy2, lineThickness, bold, textSize>(notes)

Set a tuplet.


  • Parameters (in the order of their default positions):

    • format - meter format

      • standard values ([leftBrace][numerator]:[denominator][rightBrace]) :
        • "-3-"
        • "-3:5-"
        • "-2:5"
      • default value: ""

    • dy1, dy2 - graphic vertical offsets of the two ends of the tuplet's line (defines tuplet's line slope)

      • standard values:
        • unit parameters
      • default value: 0

    • lineThickness - graphic thickness of tuplet's line

      • standard values:
        • unit parameter
      • default value: 1

    • bold - defines if the text is bold or not

      • standard values:
        • "true"
        • "false"
      • default value: "false"

    • textSize - graphic size of the text

      • standard values:
        • unit parameter
      • default value: 1


  • Semantics:
    • A tuplet corresponds to any rhythm that involves dividing the beat into a different number of equal subdivisions from that usually permitted by the time-signature.

Examples: 

[ \tuplet<"-3-", textSize=1.5, bold="true", lineThickness=0.2, dy1=10> (a a a) ]
[ \tuplet<"2:5-", dy1=7, dy2=9> (g b b) ]
