Javascript GUIDO Engine
The Guido Engine can now be embedded in a web page.
It is available as a Javascript library and operates with very good performances.
Several technologies are available for the graphic rendering :
  • SVG export of the music score.
  • Use of HTML Canvas
  • Use of the Guido binary export device
  • Use of WebComponents
The Guido Engine library is an open source library for the layout and the graphic rendering of music scores developped and maintained by the Grame Computer Music Research Lab.
