This document is a simple example of web component. You have to run it with a http server because template is in an external file and most of browser doesn't allow cross browsing domain.

You can use the python SimpleHttpServer on linux. (type "python -m SimpleHTTPServer 8000" in a console)

Your browser must support import files.
